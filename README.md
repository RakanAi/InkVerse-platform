## Project Description
InkVerse is a full-stack web application that allows users to browse, read, and manage original stories and fanfiction content. The platform focuses on clean UI design, scalable backend architecture, and real-world deployment.

## Technologies Used
Frontend:
- React
- HTML5, CSS3, Bootstrap
- Axios

Backend:
- ASP.NET Core Web API
- C#
- Entity Framework Core
- JWT Authentication

Database:
- SQL-based relational database

Hosting:
- SmarterASP

## Architecture
The frontend communicates with the backend via RESTful API endpoints. Business logic is handled in a service layer, and data access is managed using Entity Framework Core.

## How to Run Locally
1. Run the ASP.NET Core backend
2. Configure the database connection
3. Run the React frontend using npm
4. Access the application via browser
